## Objectives
Quantify the cleaned batch 4 peptide digests and prepare injections for LC-MS, targeting 400 ng
on column per sample.

## Experimental Summary
| Parameter | Description | This experiment |
|---|---|---|
| Sample type | what was measured | tf-scan atlas, batch 4 digests |
| Num. samples | total, including controls | 3 (TB-0431, TB-0432, TB-0433) |
| Target load | on-column amount | 400 ng per sample |

## Procedure (as performed)
- Resuspended each dried digest in 20 µL 0.1% FA.
- Quantified peptide by colorimetric assay (Pierce Quantitative Colorimetric Peptide Assay). The
  neat stocks read above the top of the assay's linear range, so each was quantified from a 1:10
  dilution in 0.1% FA and read against the standard curve.
- Prepared each sample for injection to load 400 ng on column.

## Results
- Peptide concentrations (colorimetric):
  - TB-0431: 0.24 µg/µL
  - TB-0432: 0.20 µg/µL
  - TB-0433: 0.28 µg/µL
- Injection for TB-0432: 2 µL loads 400 ng on column (0.20 µg/µL × 2 µL = 0.4 µg). Other samples
  scaled to 400 ng the same way.

None.

Batch 4 quantified and prepared at 400 ng on column; ready for acquisition. *(LP)*
